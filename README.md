# WalkerRemoteBoard Rev5 as an ESP8266 Synth 🎹🎛️⚡

This repository contains experimental sketches and documentation for using the **WalkerRemoteBoard Rev5** as a small digital synth and frequency generator based on the **ESP8266**.

The goal of this project is to explore how electrical frequencies, PWM signals and software-generated waveforms can be used to create sounds, rhythms and simple synth effects with a microcontroller.

---

## Overview

The **WalkerRemoteBoard Rev5** is an experimental board based on the **ESP8266**.  
In this repository, the focus is not on remote control, motors or sensors, but exclusively on:

- sound generation;
- electrical frequencies;
- PWM audio experiments;
- simple digital synth effects;
- arcade-style sounds;
- rhythm and drum patterns.

The same hardware can behave in many different ways depending on the sketch uploaded to the ESP8266.

---

## ESP8266 Audio Generation

The ESP8266 integrates a **32-bit RISC Tensilica L106 processor**.

It is important to clarify that the ESP8266 does **not** include a real analog audio DAC.  
For this reason, the audio is not generated as a pure analog signal.
